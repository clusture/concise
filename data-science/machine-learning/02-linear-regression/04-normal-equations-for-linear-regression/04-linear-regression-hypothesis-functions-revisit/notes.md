# Hypothesis Function Revisit for Linear Regression

Let us recall the linear regression hypothesis function :
$$h_{\theta}(x) = \sum_{j=1}^{d} \theta_{j}x_{j} = x^{T}\theta = \theta^T x, \quad \text{predicted target variable}$$
